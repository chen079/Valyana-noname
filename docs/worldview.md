# 瓦尔亚纳世界观

《熔炉》的世界发生于一片叫做瓦尔亚那的大陆上，这里位于“奇点”宇宙中的某个半位面之中。

这是一片多种族交融之地。无数风格各异的建筑以一种不失美感的奇异方式排列在条条街道之上，你可能半分钟前还在享受地中海的明媚阳光，下一秒就进入神秘莫测的东方古国，这有时甚至会发生在一次转角之间。人们的生活方式因此复杂而充满奇异风情。

瓦尔亚那大陆上的数百个种族互相联系、互相影响。大陆东西两侧皆是深不可测的无尽海洋，未知生物出没其间，使那里成为生命禁区。

大陆北部最大的国家是克拉，兽人语中意为“家园”。它由除狼人外的其他兽人共治，是君主制国家，风貌与中世纪社会相近。

更东侧的迦奈尔联邦是兽人与人类的共同聚居地。由于禁魔矿石存在，当地人无法使用魔法，于是以科技为主要支柱，科技水平近似二战时期，生活形态却更接近现代。法师在大陆上被称为获法者，是强大而稀有的存在，而迦奈尔联邦中的大量禁魔矿石让这片土地免于魔法侵袭。

禁魔矿石如何影响魔法，精灵族大百科《瓦尔亚那预言集》已有记载。禁魔矿石周围会形成强大力场，将空间中的魔法能量压缩为密度极高的场物质，使生物无法直接使用。

向北是极北之地，冻结的巨大冰川与无尽荒芜莽原相连，只有少数雪人族居住。向南跨过人鱼之海，外圈是茫茫苔原，中心则接近任何生物都未曾进入过的极点，接近零下100℃的寒冷足以让大部分生物瞬间化作冰雕。

大陆中部定居着数个人类国家，战乱与纷争是这里的主旋律。七千年前的“万灵之森战役”曾以人类的颓败与生命之树的坍塌告终，人类元气大伤，精灵族也几乎失去繁育后代的能力。

大陆中部还有被称为深渊的区域，居住着除吸血鬼以外的大部分黑暗生物。它屡遭讨伐，却也不断滋生出新的黑暗。

作为生活在大陆上的人们，下一顿吃什么往往比大陆本身更重要。你可以与矮人或侏儒交易，也可以寻找精灵踪迹，前往深渊或天界，见识魔法、科技与各种未知生物。

总之，瓦尔亚那是一个能容纳一切想象的世界。
